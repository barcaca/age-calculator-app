# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Purple: hsl(259, 100%, 65%) violet-500
- Light red: hsl(0, 100%, 67%) rose-500

### Neutral

- White: hsl(0, 0%, 100%) white
- Off white: hsl(0, 0%, 94%) zinc-100
- Light grey: hsl(0, 0%, 86%) zinc-300
- Smokey grey: hsl(0, 1%, 44%) neutral-500
- Off black: hsl(0, 0%, 8%) neutral-900

## Typography

### Body Copy

- Font size (inputs): 32px

### Font

- Family: [Poppins](https://fonts.google.com/specimen/Poppins)
- Weights: 400i, 700, 800i
